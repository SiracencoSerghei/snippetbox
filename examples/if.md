## Початковий оператор блоку If

Умовний ifоператор може мати "початковий" оператор. Змінна(и), створена(і) в початковому операторі, визначена(і) лише в межах тіла оператора if.

```
if INITIAL_STATEMENT; CONDITION {
}
```

### Чому я маю це використовувати?
 **Він має дві цінні мети:**

- Це трохи коротше.
- Це обмежує область дії ініціалізованих змінних ifблоком

*Наприклад, замість того, щоб писати:*
```
length := getLength(email)
if length < 1 {
    fmt.Println("Email is invalid")
}
```
*Ми можемо зробити:*
```
if length := getLength(email); length < 1 {
    fmt.Println("Email is invalid")
}
```
У наведеному вище прикладі length недоступний у батьківській області видимості, що зручно, оскільки він нам там не потрібен — ми випадково не використаємо його деінде у функції.
